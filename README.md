# Research Trend Analysis System (TrendTrack)

A system for analyzing scientific publication trends.

## 🤖 AI Assistant Context & Guidelines

Welcome! If you are an AI assistant reading this, this section is intended to give you the context of the project to save tokens and time.

### Project Structure

This repository contains both the Frontend and Backend for the Research Trend Analysis System.

*   **Frontend (Current Directory)**: 
    *   The root directory contains the Frontend code.
    *   It is a React application using Vite (`src/`, `vite.config.js`, `package.json`).
    *   Key directories: `src/` (contains components, pages like `src/Page/Detail/PaperDetail/PaperDetail.jsx`, and styles).
*   **Backend**:
    *   The Backend code is located in the subdirectory: `Publication-Trend-Tracking-System-develop/`
    *   When working on backend tasks, please change your focus to this directory.

### General Guidelines for AI
